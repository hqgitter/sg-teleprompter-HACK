The Teleprompter Bot!

This bot is used to help users learn via flashcards utilising
a platform that would normally be used for chatting.
As such, users would be able to access their flashcards as often as they choose
to talk on such a platform, thus providing use to the flashcard.

To use the bot you can talk to it via 
t.me/SgTelePrompterBot
if you have a telegram account.

Available commands are:
/flashcard
/new
/question
